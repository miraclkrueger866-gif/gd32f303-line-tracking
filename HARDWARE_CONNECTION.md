# 硬件连接说明

## 🔌 详细接线图

### 1. 电磁传感器接线

```
         GD32F303
         --------
     
PA0 (ADC10) ←── LL 传感器 (左左)
PA1 (ADC11) ←── LM 传感器 (左中)
PA2 (ADC12) ←── MM 传感器 (中)
PA3 (ADC13) ←── RM 传感器 (右中)
PA5 (ADC15) ←── RR 传感器 (右右)

GND ←── 所有传感器GND
3.3V ←── 所有传感器VCC
```

### 2. 电机驱动接线

```
左电机连接：
PB0 (IN1) → L298N输入1
PB1 (IN2) → L298N输入2
OUT1, OUT2 → 左电机

右电机连接：
PB2 (IN3) → L298N输入3
PB3 (IN4) → L298N输入4
OUT3, OUT4 → 右电机

电源：
GND → L298N GND
+5V → L298N +5V
```

### 3. 编码器接线（可选）

```
左编码器：
PA6 (TIMER2_CH0) ← 编码器A相
PA7 (TIMER2_CH1) ← 编码器B相

右编码器：
PA0 (TIMER1_CH0) ← 编码器A相
PA1 (TIMER1_CH1) ← 编码器B相

GND ← 编码器GND
+5V ← 编码器VCC (如需要)
```

### 4. OLED显示屏接线

```
PA11 → SDA (数据线)
PA12 → SCL (时钟线)
GND  → GND
3.3V → VCC
```

### 5. RGB LED接线

```
PB12 → DIN (数据输入)
GND  → GND
+5V  → VCC (通常需要独立供电)
```

### 6. 蜂鸣器接线

```
PA6 (TIMER0_CH0) → 正极
GND → 负极（通过470Ω保护电阻）
```

### 7. 按键接线

```
K1: PA4  ← 按键
K2: PA11 ← 按键
K3: PB0  ← 按键
K4: PB1  ← 按键
K5: PC12 ← 按键
K6: PD2  ← 按键

GND ← 所有按键另一端
```

### 8. 串口通信（可选）

```
USART0：
PA9  (TX) → USB转串口模块 RX
PA10 (RX) → USB转串口模块 TX

USART1：
PA2 (TX) → 蓝牙模块 RX
PA3 (RX) → 蓝牙模块 TX

GND ← 模块GND
```

## ⚡ 电源分配

```
主电源（5V）：
├─ L298N电机驱动模块
├─ RGB LED
└─ GD32F303通过3.3V稳压

3.3V电源：
├─ GD32F303核心板
├─ 所有传感器
├─ OLED显示屏
└─ 按键

推荐配置：
- 电池：2S LiPo（7.4V）或 4×AA电池（4.8V）
- 电流：建议5A以上电源
```

## 🚨 常见问题

### Q: OLED显示不出来
**检查**：I2C接线是否正确（PA11/PA12）

### Q: 电机不转
**检查**：L298N电源是否接入，控制引脚是否正确

### Q: 传感器读数异常
**检查**：ADC引脚接线，传感器VCC和GND是否正确

### Q: 按键无响应
**检查**：按键另一端是否都连接到GND

### Q: RGB LED不亮
**检查**：独立5V电源是否接入，数据线是否正确

## 📐 PCB设计建议（如需）

### 布线原则
- 模拟信号（传感器）远离数字信号（电机控制）
- 电源线用粗线（≥1mm）
- 所有模块的GND在一点汇集
- I2C总线加100Ω终端电阻

### 层数建议
- 最少2层：信号层 + GND层
- 推荐4层：VCC、GND、信号1、信号2

## 🎯 调试顺序

1. **硬件检查** - 确保所有接线正确
2. **传感器测试** - 用串口输出ADC值
3. **电机测试** - 分别控制左右电机
4. **集成测试** - 开启完整巡线程序
5. **参数调优** - 按照PARAMETER_TUNING.md调优
